# Як грати — Apartment Anomaly Hotline

## Швидкий старт

1. Встанови [Godot 4.3+](https://godotengine.org/download)
2. Відкрий папку проєкту → **Import** → **Import & Edit**
3. Натисни **F5** (Play)
4. **Start Night 1** → почни зміну

## Керування

| Дія | Як |
|-----|-----|
| Відповісти на дзвінок | **Answer call** |
| Питання абоненту | Кнопки Ask location / appearance / … |
| Камери CCTV | **< CAM** / **CAM >** |
| Рішення | Dispatch, Lock floor, Ignore, … |
| Пауза | **Esc** |

## Налаштування (Settings)

- **Language** — English / Українська (за замовчуванням UK)
- **Caller voice (VO)** — озвучка дзвінків (178 записів у проєкті)
- **Subtitles** — субтитри реплік
- Гучність: Master, SFX, Music, Voice

## Мета зміни

- Пережити ніч (~10 хв)
- Слухати абонентів, дивитись **CCTV**, читати **Internal note**
- Обрати правильний протокол
- Тримати **ANOMALY %** низьким

## Коли дзвінки закінчились

Після ~7 дзвінків черга порожня (**Черга: завершено**). Далі:

- Таймер зміни (**ЗМІНА**) йде далі — стеж за CCTV
- **Аномалія повільно росте** навіть без дзвінків
- Дочекайся **00:00** — залежно від % буде кінцівка

## Кінцівки зміни

| Умова | Результат |
|--------|-----------|
| **Аномалія ≥ 90%** | Колапс (програш) — зміна одразу закінчується |
| **Аномалія ≤ 15%** після всіх дзвінків | Успішне стримування (можливо раніше) |
| **Таймер 0:00**, аномалія &lt; 40% | Вижив |
| **Таймер 0:00**, аномалія ≥ 70% | Колапс |
| Інакше на 0:00 | Нейтральне виживання |

**80%+** — попередження в журналі. **90%+** — game over.

## Поради

1. Завжди **Answer call** перед dispatch
2. Перемкни камеру з підказки в Internal note
3. Mimic / UNVERIFIED — не довіряй наказам «скасувати тривогу»
4. Побутові дзвінки (протікання, шум) — ignore або dispatch, не cut power на весь будинок

## Збірка .exe / .app (опційно)

У Godot: **Project → Export** → додай **Windows/macOS/Linux** preset → Export.

